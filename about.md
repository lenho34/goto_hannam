# Abandoned Places Blog

This blog explores the current state of places that were once beloved by many people around the world.

While places change over time, the memories and stories remain.

## Tech Stack

- **Jekyll**: Static site generator
- **GitHub Pages**: Hosting
- **Google Custom Search API**: Image search
- **GitHub Actions**: Automation
